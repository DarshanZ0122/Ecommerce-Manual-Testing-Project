# E-Commerce Manual Testing Practice Project

## Project URL
https://sauce-demo.myshopify.com/

## Project Overview
This repository contains end-to-end manual testing documentation for a basic 
E-commerce web application. The project covers functional, UI, usability, 
cross-browser, responsive, accessibility, and basic security testing.

## Scope of Testing
- User Registration & Login
- Product Catalog & Search
- Product Details Page
- Cart & Checkout
- Order History
- Profile Management

## Testing Types Covered
- Functional Testing
- Smoke & Regression Testing
- UI/UX Testing
- Cross-Browser Testing
- Responsive Testing
- Negative & Boundary Testing
- Accessibility Testing (Basic)
- Security Testing (Basic)

## Deliverables
- Project Overview
- Functional Requirements Document (FRD)
- Test Plan & Strategy
- Test Scenarios
- 150+ Manual Test Cases
- Defect Report & Defect Summary
- RTM
- Test Summary Report

## Tools Used
- Manual Testing
- MS Excel/ TestRail
- Jira (Practice)
- MSSQL 

## Note
This is a practice project created for learning and portfolio purposes.
